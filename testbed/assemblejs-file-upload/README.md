# AssembleJS File Upload Cookbook

This project demonstrates file upload implementations with progress tracking and multi-file support in AssembleJS.

## Features

- Single and multiple file uploads
- Drag and drop file interface
- Upload progress tracking
- File type validation
- Image previews and thumbnails
- File size limits and chunking
- Server-side file processing
- Secure upload handling

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Your application will be available at http://localhost:3000 by default.
