## Features

- Responsive layouts
- Mobile-first design
- Adaptive components
- Media queries
- Flexible images and media
- Responsive typography
- Device detection
- Touch-friendly interfaces

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Your application will be available at http://localhost:3000 by default.
